# Current AI Facts (last verified: April 2026)

Update this file whenever web search reveals changes. This prevents stale claims in articles.

## Claude (Anthropic)

- **Models:** Opus 4.6 (flagship), Sonnet 4.6 (fast)
- **Context window:** 1M tokens (~750K words) on Opus 4.6
- **Pricing:** Free / Pro $20/mo (~$17 annual) / Max $100-200/mo / Team $25/user/mo / Enterprise custom
- **Free plan includes:** Sonnet 4.5, 30-100 messages/day (5h rolling window), up to 5 Projects, file uploads (20 files, 30MB), Memory, Artifacts, extended thinking (limited), web search (basic)
- **Pro adds:** 5x usage, all models incl. Opus 4.6, unlimited Projects, Styles, priority access, full extended/adaptive thinking
- **Projects:** Persistent workspaces with custom instructions + uploaded files. Available on all plans (5 on free, unlimited on Pro). Shared on Team plan.
- **Memory:** Cross-conversation memory on all plans including free (since March 2026)
- **Artifacts:** Interactive documents, downloadable .pptx/.docx — available on ALL plans including free
- **Extended/Adaptive thinking:** Extended thinking deprecated on Opus 4.6/Sonnet 4.6, replaced by adaptive thinking (Claude auto-decides thinking depth). Available on all plans with limited usage on free.
- **Web search:** Claude HAS basic web search. Not as deep as ChatGPT's Deep Research, but it can look things up.
- **Styles:** Preset and custom writing styles. Pro and above.

## ChatGPT (OpenAI)

- **Models:** GPT-5.4 Thinking (flagship)
- **Context window:** 272K tokens standard interface. 1M via API/Codex.
- **Pricing:** Free / Go $8/mo / Plus $20/mo / Pro $200/mo / Business $25/user/mo / Enterprise custom
- **Plus includes:** GPT-5.4 Thinking, Deep Research (10 runs/mo), Sora video, DALL-E images, Codex coding agent, Agent Mode, Custom GPTs, ~180 images/day, ad-free
- **Deep Research:** Web browsing + multi-source synthesis. Stronger than Claude's basic search.
- **Projects:** ChatGPT has Projects too (conversation folders). Less powerful than Claude's — no persistent file knowledge base in the same way.
- **Custom GPTs:** Single-task tools, not persistent workspaces like Claude Projects.

## Gemini (Google)

- **Models:** Gemini 3 Pro, Gemini 2.5 Pro
- **Context window:** 1M tokens
- **Pricing:** Free / AI Pro ~$20/mo / AI Ultra ~$30/mo
- **Key advantage:** Native Google Workspace integration (Gmail, Docs, Sheets, Drive)
- **Includes:** 2TB Google One storage with AI Pro
- **Image generation:** Imagen (less capable than DALL-E)
- **Writing quality:** Weakest of the three for nuanced professional writing

## Common Mistakes to Avoid

- DON'T say "Claude doesn't have web search" — it does (basic)
- DON'T say "Artifacts require Pro" — they're on free since late 2025
- DON'T say "ChatGPT's context is 128K" — it's 272K standard (1M API)
- DON'T say "ChatGPT doesn't have Projects" — it does, they're just weaker
- DON'T say "extended thinking" without noting it's deprecated in favor of adaptive thinking
- DON'T say Claude's context is "200K words" — it's ~750K words (1M tokens)
